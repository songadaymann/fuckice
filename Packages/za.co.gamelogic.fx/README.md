# Gamelogic Fx
This package contains code and tools for developing basic visual effects.

## Where are the samples?
You can install samples uysing the package manager.
1. Open the Package Manager window (`Window > Package Manager`).
2. Select `Unity Registry` from the dropdown at the top left.
2. Select the `Gamelogic Fx Dithering` package from the list on the left.
4. In older versions of Unity you will see the sammples listed at the bottom right of the window. In newer versions,
	click on the `Samples` tab at the top center of the window. Click on the sample you want to install. (Check if one
	of the samples is not a requirement for other samples, and if so install it first.)
